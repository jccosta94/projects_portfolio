# Skoda AI Ops — Content Automation Pipeline

🚧 **Case study in progress.** Full architecture writeup landing soon.

Multi-channel content automation pipeline for a referral automotive brand. Handles **social media production**, **blog generation**, **Veo video workflows**, and **publishing automation across channels**. Production traffic to a real audience.

**What's documented here when the writeup lands:**
- Architecture overview — orchestration layer, content pipeline, publishing workflow, media generation flow, deployment topology
- Workflows — blog generation, social post pipeline, Veo video workflow, approval routing, publishing orchestration
- Orchestration — MCP integration, agent routing, scheduling logic, model selection per content type
- Case study — business problem, architecture evolution, orchestration design, AI stack, cost routing, lessons learned
- Sample prompts + partial code examples

**Architecture concerns this addresses:**
- Multi-channel content fan-out from a single brief
- Veo video generation orchestration
- Brand-voice consistency across formats (blog vs social vs video)
- Time-zone-aware scheduled publishing across platforms
- Approval routing with human-in-the-loop gate before public release
- Cost-aware model selection (which content needs Opus, which can use cheaper models)
