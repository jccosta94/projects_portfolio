# Salesforce Delivery Copilot

🚧 **Case study in progress.** Full architecture writeup landing soon.

An AI copilot for Salesforce delivery teams. Takes meeting transcripts → produces structured notes → outputs process flows and user stories → keeps the project plan, phases, deliverables, and context up-to-date as the project evolves. **Single input becomes a coherent multi-artifact output**: notes + process diagrams + user stories + a continuously-updated project plan, cross-referenced and self-consistent. Production-deployed for Salesforce delivery work.
